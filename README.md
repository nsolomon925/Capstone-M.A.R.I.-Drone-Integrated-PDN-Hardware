# Capstone-M.A.R.I.-Drone-Integrated-PDN-Hardware
Integrated 3-stage PDN for the M.A.R.I. Drone. Features a hierarchical 14.8V→5V→3.3V architecture to isolate 71A+ motor transients from sensitive analog sensors. Includes a custom LED-based battery monitoring system for real-time safety alerts. Validated via Mathcad dynamic power modeling and comprehensive current-draw analysis.
